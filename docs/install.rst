Installation
============

.. _requirements-to-use:

Requirements
============

Installing from source
----------------------

.. code-block:: console

    $ git clone https://github.com/devanshshukla99/sphinx-versioned-docs
    $ cd sphinx-versioned-docs
    $ pip install .


Installing from PyPi
--------------------

.. code-block:: bash

    pip install sphinx-versioned-docs
